A developer can set the same/multiple tags to multiple views for dynamic view hiding functionality as below:


[block:code]
{
  "codes": [
    {
      "code": "yourView.tag = \"YOUR TAG VALUE\";",
      "language": "swift"
    }
  ]
}
[/block]

Alternatively you can set tags in StoryBoard as well.
